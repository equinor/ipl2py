ipl2py
======

ipl2py is a source-to-source compiler. It compiles the 
`RMS <https://www.aspentech.com/en/products/sse/aspen-rms>`_ IPL scripting
language into Python compatible with the RMS Python environment.


Quick start
-----------

.. toctree::
    :caption: Quick start

.. code-block:: sh

    git clone git@github.com:equinor/ipl2py.git
    cd ipl2py
    pip install .
    ipl2py script.py
